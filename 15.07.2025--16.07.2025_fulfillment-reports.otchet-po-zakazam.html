Про отчёт по продажам маркированных товаров рассказываем в отдельной статье

Из отчёта по заказам вы можете узнать:

  • общие сведения о ваших заказах — к примеру, когда и какие товары были куплены;

  • аналитическую информацию о заказах — например, стоимость доставки и способ оплаты заказа покупателем;

  • информацию о кластерах доставки — чтобы вовремя заполнить статистическую форму при продаже в страны ЕАЭС.

По умолчанию отражаем в отчёте общие сведения о ваших заказах. Чтобы скачать расширенную версию, отметьте в Настройках отчётов дополнительную информацию, которую нужно добавить в файл:

  1. аналитические данные — про доставку, оплату и статус Premium у покупателя;

  2. данные покупателя (только для realFBS) — про имя покупателя и адрес доставки;

  3. данные с маркировки (только для FBS) — про информацию на этикетке;

  4. штрихкод для ювелирных изделий (только для FBO) — про штрихкоды ювелирных изделий, которые вы поставили. Подробнее о поставках ювелирных изделий на FBO 

Как скачать отчёт по заказам

В личном кабинете можно скачать отчёты с данными не более чем за 3 месяца. Если вам нужны данные за больший период, скачайте несколько отчётов.

Для этого откройте раздел Аналитика → Отчёты → Заказы. Там можно выбрать отчёты в зависимости от схемы работы:

  • Заказы с моих складов — в отчёте отражаются данные о товарах на складе продавца (FBS и realFBS).

  • Заказы со складов Ozon — в отчёте отражаются данные о товарах на складе Ozon (FBO).

Для продавцов на схеме FBS или realFBS

  1. Откройте раздел Аналитика → Отчёты → Заказы → Заказы с моих складов.

  2. Нажмите Настройки отчёта и выберите дополнительные данные, которые вам нужны в отчёте:

    • аналитические данные;

    • данные с маркировки;

    • данные покупателя (только для realFBS).

    Если вы не поставите флажок рядом с какими-то из этих пунктов, в отчёте будет отражена только основная информация.

  3. Настройте фильтры:

    • по дате;

    • по артикулу продавца;

    • по номеру отправления;

    • по названию продукта.

  4. Выберите период, за который хотите скачать отчёт — сегодня, 3 дня, 7 дней, 14 дней, 3 месяца или свой вариант в календаре.

  5. Нажмите Скачать. Загрузим данные в формате CSV на ваше устройство. Что делать, если неправильно открывается CSV-файл

Для продавцов на схеме FBO

  1. Нажмите Аналитика → Отчёты → Заказы → Заказы со складов Ozon.

  2. Нажмите Настройки отчёта и выберите дополнительные данные, которые вам нужны в отчёте:

    • аналитические данные;

    • штрихкод для ювелирных изделий.

    Если вы не поставите флажок рядом с какими-то из этих пунктов, в отчёте будет отражена только основная информация.

  3. Настройте фильтры:

    • по дате;

    • по артикулу продавца;

    • по номеру отправления;

    • по названию продукта.

  4. Выберите период, за который хотите скачать отчёт — сегодня, 3 дня, 7 дней, 14 дней, 3 месяца или свой вариант в календаре.

  5. Нажмите Скачать. Загрузим данные в формате CSV на ваше устройство. Что делать, если неправильно открывается CSV-файл

Что содержит отчёт по заказам

  • Номер заказа — который сделал покупатель.

  • Номер отправления — номер в формате 0000000000-0000-0. Первые 10 цифр привязаны к покупателю, следующие 4 показывают, какой по счёту заказ у этого покупателя, а последняя цифра — количество упаковок или товаров в одном заказе.Поможет разобраться, если заказ состоял из нескольких посылок.

  • Принят в обработку — дата и время поступления заказа в ваш личный кабинет.

  • Дата отгрузки — когда отправление должны передать в доставку. Эта дата рассчитана системой и иногда может не совпадать с фактической датой передачи в доставку. 

  • Статус отправления — что с заказом на момент загрузки отчёта:

    • доставлен;

    • доставляется;

    • ожидает отгрузки;

    • ожидает сборки;

    • отменён;

    • открыт спор;

    • не принят.

    Подробнее о статусах отправления при продаже со своего склада

  • Дата доставки — когда заказ доставили покупателю.

  • Фактическая дата передачи в доставку — дата, когда заказ был отгружен со склада и передан в доставку.

  • Сумма отправления — стоимость товара с учётом количества товара (Сумма отправления = Итоговая стоимость товара × Количество товаров в отправлении).

  • Код валюты отправления — по которой купили товар.

  • Наименование товара — название товара как в карточке.

  • Ozon ID — идентификатор товара на Ozon. Его можно найти в отчёте по товарам в разделе Аналитика → Отчеты → Товары или в карточке товара в личном кабинете — справа от названия в формате «Код: ххх».

  • Артикул — ваш артикул товара.

  • Итоговая стоимость товара — цена с учётом всех акций и скидок.

  • Код валюты товара — по которой товар продаётся.

  • Стоимость товара для покупателя — цена с учётом скидок, по которой покупатель приобрёл товар.

  • Количество товаров — сколько штук товара в отправлении.

  • Стоимость товара для покупателя — цена, по которой покупатель купил товар. Рассчитывается с учётом скидок за баллы, скидки по карте Ozon, программ лояльности партнёров и региональных коэффициентов.

  • Стоимость доставки — сумма за доставку товара до покупателя.

  • Связанные отправления — укажем их, если разделили вашу заявку на несколько поставок.

  • Цена товара до скидок — сколько стоил товар до всех скидок.

  • Скидка % — сколько процентов составила скидка.

  • Скидка руб — сумма скидки в рублях.

  • Акции — по какой акции заказали товар.

  • Подъём на этаж — стоимость подъёма крупногабаритного товара на этаж покупателя.

  • Объёмный вес товаров, кг (для FBO) — какой объёмный вес у заказанного товара в килограммах.

Что делать, если неправильно открывается CSV-файл

Формируем CSV-файл с данными в виде простого текста. Форматирование накладывается в той программе, с помощью которой вы открываете файл. Иногда из-за различия кодировок CSV файл может неправильно открыться в Excel. В таких случаях импортируйте данные из текста в Excel, тогда программа автоматически определит тип кодировки, и файл откроется правильно. Для этого:

  1. Создайте новый документ в Excel.

  2. Перейдите на вкладку Данные → Из текста.

  3. Выберите нужный отчёт и нажмите Получить данные.

  4. Выберите формат данных С разделителями и нажмите Далее.

  5. Выберите разделитель Точка с запятой и нажмите Далее.

  6. Нажмите Дополнительно, выберите Десятичный разделитель «.» и нажмите Ок. 

  7. После этого нажмите Готово.

  8. Импортируйте данные на нужный лист.
